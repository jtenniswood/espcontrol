# 3.5-inch JC3248W535

![EspControl on the 3.5-inch JC3248W535: landscape home screen with temperature, clock, and control tiles](/images/jc3248w535-hero.jpg)

The **Guition JC3248W535** is a compact 3.5-inch landscape touchscreen powered by an **ESP32-S3** processor. It connects via QSPI for fast display updates and includes an optional LiPo battery connector for portable use. The home screen fits **6 cards** in a 3×2 grid.

## Specifications

| | |
|---|---|
| **Screen size** | 3.5 inches |
| **Resolution** | 480 × 320 |
| **Orientation** | Landscape |
| **Display interface** | QSPI (mipi_spi) |
| **Processor** | ESP32-S3 |
| **WiFi** | Built-in (2.4 GHz) |
| **PSRAM** | Octal, 80 MHz |
| **Touch** | AXS15231B capacitive (I2C) |
| **Power** | USB-C |
| **Battery** | LiPo (optional) |

For a full walkthrough including WiFi setup and Home Assistant pairing, see the [Install guide](/getting-started/install).

::: tip Deep-sleep reset
This device enters deep sleep on software reset to reinitialize the QSPI display. If the screen is unresponsive after flashing, unplug the USB-C cable and reconnect it — this triggers a full hardware power cycle.
:::

## ESPHome Manual Setup

If you use ESPHome and prefer to compile firmware yourself:

```yaml
substitutions:
  name: "desk-screen"
  friendly_name: "Desk Screen"

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

packages:
  setup:
    url: https://github.com/jtenniswood/espcontrol/
    file: devices/guition-esp32-s3-jc3248w535/packages.yaml
    refresh: 1sec
```
