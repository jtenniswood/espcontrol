# Appearance

These settings control the colours your buttons use to show whether a device is on or off, and the colour used for sensor cards. You'll find them in the **Settings** tab on the [Setup](/features/setup) page, under the **Appearance** section.

- **On colour** — the colour buttons show when a device is switched on. Use the colour picker or type a colour code (for example, `FF8C00` for orange).
- **Off colour** — the colour buttons show when a device is switched off.
- **Sensor colour** — the background colour used for sensor cards.
